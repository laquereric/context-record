VV Ontology — biological namespace

Namespace URI: https://verticalvertical.net/ns/biological#

This namespace defines the shared vocabulary used by context-record and all PostCentaur domain ontologies.

Available formats

FormatMIME typeFile
JSON-LD Context application/ld+json context.jsonld
Turtle Ontology text/turtle vv-ontology.ttl

Usage

Reference the JSON-LD context in any document:

{
  "@context": "https://verticalvertical.net/ns/biological#",
  "@type": "Entity",
  "hasFeature": "weather-sealed"
}

Content negotiation

GitHub Pages does not support server-side content negotiation. Use the direct file links above, or fetch by format: 

# JSON-LD context
curl https://vertical-vertical.github.io/context-record/ns/biological/context.jsonld

# Turtle ontology
curl https://vertical-vertical.github.io/context-record/ns/biological/vv-ontology.ttl

Once DNS for verticalvertical.net is configured with a reverse proxy (nginx or Cloudflare Workers), true Accept-header content negotiation will be available at the namespace URI.
